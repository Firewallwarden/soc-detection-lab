# SOC Detection Lab

## Overview

This project documents the design, deployment, and operation of a home Security Operations Center (SOC) lab. The goal is to gain hands-on experience with SIEM deployment, endpoint monitoring, threat detection, and incident response using enterprise security tools.

## Project Goals

- Deploy a Wazuh SIEM server
- Monitor Windows and Linux endpoints
- Generate and investigate security events
- Create custom detection rules
- Document incident response procedures
- Build a professional cybersecurity portfolio project
